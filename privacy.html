Privacy Policy

Last updated: 28 June 2026

ml-affilliate-engine is a local single-operator tool ("Solopreneur tool") that automates the publication of Mercado Libre affiliate carousel posts to the operator's own TikTok account via the TikTok Content Posting API.

Data we access

  • TikTok account data: the operator accessed only after explicit OAuth authorisation. We request user.info.basic and video.publish them solely to publish the operator's posts.
  • Mercado Libre catalog data: publi (product title, price, image URL) is fetched from Mercado Libre's public API to populate affiliate posts.
