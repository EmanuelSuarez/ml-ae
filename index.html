ml-affilliate-engine

Local Solopreneur tool that automates Mercado Libre affiliate carousel posts to TikTok via the Content Posting API.

How it works

  1. Run locally with python -m streamlit run app.py
  2. Add a TikTok account and an affiliate campaign with Mercado Libre links
  3. The orchestrator enriches products, drafts copy (with your approval), generates the cover + product overlays, and publishes via the TikTok Content Posting API
